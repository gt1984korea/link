# 새 구절 알림(웹 푸시) 설정 가이드

아이폰/안드로이드 홈 화면 아이콘에 **빨간 배지**를 띄우고, 새 암송 구절이 등록되면
**알림**을 보내는 기능입니다. 코드는 모두 준비됐고, 아래 4단계 "한 번만" 설정이 필요합니다.

> ⚠️ 핵심 전제
> - **아이폰은 홈 화면에 추가된 "바로가기 앱"에서 열어야** 알림이 됩니다(사파리 탭 ❌, iOS 16.4+).
> - 교인 각자가 앱에서 **"새 구절 알림 받기"를 눌러 알림을 허용**해야 그 사람에게 알림이 갑니다.

---

## 1단계 — Firebase 요금제를 Blaze로 전환

Cloud Functions를 쓰려면 종량제(Blaze)가 필요합니다. (소규모 교회 사용량은 **무료 한도 내**라 실제 비용은 거의 0원이지만, 카드 등록이 필요합니다.)

1. [Firebase Console](https://console.firebase.google.com/project/victorychurch-665a9/usage/details) 접속
2. 좌측 하단 **요금제 업그레이드 → Blaze** 선택, 결제 정보 등록
3. (선택) 예산 알림(Budget alert)을 $1 등으로 설정해 두면 안심

## 2단계 — 웹 푸시 키(VAPID) 생성 후 push.js에 붙여넣기

1. Firebase Console → **프로젝트 설정(⚙️) → 클라우드 메시징(Cloud Messaging)** 탭
2. **웹 푸시 인증서(Web Push certificates)** 항목에서 **키 쌍 생성** 클릭
3. 생성된 **키 쌍(공개 키)** 문자열을 복사
4. [`push.js`](push.js)를 열어 아래 값을 교체:

   ```js
   const VAPID_PUBLIC_KEY = '여기에_복사한_공개키_붙여넣기';
   ```

   > 이 공개 키는 외부에 노출돼도 안전합니다(클라이언트 키).

## 3단계 — 배포

PowerShell에서 프로젝트 루트(`link/`)에서:

```powershell
# 함수 의존성 설치 (최초 1회)
cd functions; npm install; cd ..

# 규칙 + 함수 + 호스팅 한 번에 배포
firebase deploy --only firestore:rules,functions,hosting
```

- 처음 함수 배포 시 "필요한 API 사용 설정" 안내가 나오면 **Y**로 진행하세요.
  (Cloud Functions, Cloud Build, Artifact Registry, Cloud Messaging API 자동 활성화)

## 4단계 — 동작 확인 (아이폰 기준)

1. 사파리로 사이트를 연 뒤 **공유 → 홈 화면에 추가**로 앱 설치
2. **홈 화면의 그 앱 아이콘**으로 다시 열기 (← 반드시 설치된 앱에서)
3. **"새 구절 알림 받기"** 버튼 탭 → **허용**
4. `/admin`에서 **새 암송 구절을 등록**
5. 잠시 뒤 알림이 오고, 앱을 닫아두면 **홈 아이콘에 빨간 배지** 표시
6. 앱을 다시 열면 배지가 자동으로 사라짐

---

## 구성 요소 (참고)

| 파일 | 역할 |
|------|------|
| [`push.js`](push.js) | "알림 받기" 버튼 제어, 권한 요청, 토큰을 `pushTokens`에 저장 |
| [`sw.js`](sw.js) | 백그라운드 푸시 수신 → 알림 표시 + 홈 아이콘 배지(setAppBadge). 정적 자산 캐싱도 겸함 |
| [`functions/index.js`](functions/index.js) | 새 구절 등록 감지 → 모든 토큰에 푸시 발송, 만료 토큰 정리 |
| `firestore.rules` | `pushTokens` 컬렉션 등록 허용(목록/삭제는 함수만) |
| `index.html` | 알림 버튼 UI + 앱 열 때 배지 지우기(`clearAppBadge`) |

## 자주 묻는 질문

- **안드로이드는요?** 별도 설치 없이도 크롬에서 동작하며, 홈 화면 추가 시 아이콘 배지까지 지원됩니다.
- **아이폰에서 버튼이 "홈 화면 추가 후 가능"으로 보여요.** 정상입니다. 사파리 탭이 아니라
  홈 화면에 추가한 앱에서 열어야 알림을 켤 수 있습니다(애플 정책).
- **비용이 걱정돼요.** 새 구절을 등록할 때만 함수가 실행되고, 발송량도 적어
  Blaze 무료 한도(월 200만 호출 등)를 넘기기 어렵습니다.
- **알림을 끄고 싶어요.** 기기 설정 → 해당 앱 → 알림 끄기. 무효 토큰은 다음 발송 때 자동 정리됩니다.
