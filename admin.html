인스타그램 게시물 (1개)

인스타그램은 자동 피드 임베드가 막혀 있어, 보여줄 게시물 1개를 직접 골라 등록합니다. 등록하면 메인 화면 "인스타그램" 섹션을 펼칠 때 그 게시물이 표시됩니다. 비우고 저장하면 안내 문구로 되돌아갑니다.

※ 공개 게시물만 표시됩니다. 인스타그램 앱/웹에서 게시물 → 공유 → "링크 복사"로 주소를 가져오세요.

열기 ↗

페이스북 게시물 (1개)

보여줄 페이스북 게시물 1개를 등록합니다. 등록하면 메인 화면 "페이스북" 섹션을 펼칠 때 그 게시물이 표시됩니다. 등록하지 않으면 페이스북 버튼을 누를 때 곧바로 페이지로 이동합니다.

※ 공개 게시물만 표시됩니다. 페이스북에서 게시물 → ⋯(더보기) → "링크 복사"로 주소를 가져오세요.

열기 ↗

최신 영상 (유튜브)

메인 화면 "유튜브 채널 → 최신 소식"에 보여줄 영상입니다. 새 설교 영상을 올린 뒤, 그 영상의 유튜브 링크를 붙여넣고 저장하세요. 비우면 자동(최신 시도)으로 되돌아갑니다.

※ 유튜브 앱/웹에서 영상 → 공유 → "링크 복사"로 주소를 가져오세요.

카드 배경 이미지 관리

사용자가 구절 카드를 공유할 때 선택할 수 있는 배경 목록입니다. 기본 제공 그라디언트와 이미지는 활성/비활성 토글만 가능하며, 직접 등록한 커스텀 이미지는 삭제도 가능합니다.

새소식 관리

메인 화면 "이번 주 성경 구절" 카드 아래에 노출되는 새소식 이미지입니다. 노출 기간(오늘 포함)에 해당하는 새소식만 표시되며, 없으면 영역 자체가 숨겨집니다. 여러 개를 등록하면 모두 표시됩니다.
